# AI SEO Blog & Content Cluster Generator for Salon
## Prompt Engineering Internship – Task 3

## Project Overview
This project demonstrates how AI can be used to generate an SEO content strategy for beauty salons and wellness businesses. The goal is to help salons improve their online visibility through content clustering, keyword planning, blog topic generation, and SEO optimization.

## Business Selected
- **Business Name:** Glow & Grace Salon
- **Industry:** Beauty & Salon Services
- **Location:** Hyderabad

## Project Objectives
- Generate SEO-friendly blog topics
- Create a content pillar and content cluster strategy
- Identify target keywords
- Improve website ranking potential
- Build an effective internal linking structure
